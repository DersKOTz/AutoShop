Валавин К.А. ИСП-211 Курсовой проект 2024
* в приложении используется база MS SQL, требующая предварительной установки и настройки приложения

Инструкция по настройке базы и приложения (выполняется системным администратором)
1) Импортировать базу AutoShop.bak, находящуюся в папке Базы в MS SQL или на хостинг
2) Подключить приложение к базе AutoShop

Инструкция по запуску приложения AutoShop.exe : 
1) Запустить приложение AutoShop.exe из папки Запуск 
2) Создать аккаунт
3) Войти в аккаунт
4) Добавить выбранный товар в корзину
5) Оформить заказ в корзине
6) Запустить файл AutoShopOrders.xlsx и проверить заказ за текущее число
